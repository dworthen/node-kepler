### Kepler

Kepler is a static website generator inspired by the ruby gem, [jekyll](http://jekyllrb.com/ "jekyll") and named after [this](http://en.wikipedia.org/wiki/Johannes_Kepler "Johannes Kepler") guy. 

#### Installing kepler

```bash
$ npm install kepler -g
```
